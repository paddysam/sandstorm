# 《末日沙尘暴》人物设定与关系网

---

## 一、主角：林阳

### 基础信息
- **年龄**：22岁（故事开篇）
- **外形**：普通身材，黑发略长，眼神开篇散漫，末日后逐渐锐利。左耳廓后期出现矿物结晶纹路（觉醒外显标志）
- **开篇状态**：住在父母家的宅男游戏迷，房间里一台3D打印机，架子上全是自己打印的模型玩具

### 身份弧线
```
普通城市青年
→ 幸存者（第23章：父母遇难）
→ 持枪者（第26章：打印手枪）
→ 四人团队核心（第30章：四人组成型）
→ 一阶觉醒者（第42章：矿物共鸣首次激活）
→ 三阶共振体（第二卷：能力全面显现，各方争夺目标）
→ 风暴意识候选（终卷）
```

### 性格弧线
- **开篇**：懒散，沉迷游戏，对世界的兴趣仅限于屏幕内
- **父母遇难后**：沉默，愤怒内化，但不外显，骨子里有一种不甘
- **四人团队期**：开始主动承担，判断力快速成长，但仍会犹豫
- **能力觉醒后**：变得笃定，开始拥有某种非人类的感知视角，与身边人的距离感悄悄拉大

### 林阳的身份秘密（核心暗线）

**逐步揭露路径：**

| 章节 | 揭露内容 | 林阳知道 | 读者知道 |
|------|---------|---------|---------|
| 第3章 | 父亲提到北方矿区旧事 | 没太在意 | 有伏笔 |
| 第5章 | 父亲上锁的抽屉：矿区工牌、旧合影、技术文件 | 感觉有隐瞒 | 父亲有秘密 |
| 第22章 | 电视10秒信号："地层链式反应" → 父亲脸色失去血色 | 父亲知道这个 | 这是人祸 |
| 第33章 | 沈雨认出工牌上的公司名，查出"地层共振能源开发" | 父亲与计划有关 | 父亲是参与者 |
| 第38章 | 沙翁："你耳朵里的频率，你父亲也能听见" | 能力可能是遗传 | 父亲与林阳能力相关 |
| 第46章 | 地下遗迹出现，矿化成分与沙暴一致 | 沙化不是第一次 | 文明循环 |
| 第50章 | 林阳"听见"正东方向父亲声音（悬念） | 父亲还存在？ | 第二卷钩子 |
| 第二卷 | 奚然给出档案：父亲是大地呼吸计划核心工程师，引爆后主动入地层试图关闭，从此失踪 | 能力是遗传，不是随机觉醒 | 父亲以矿物意识存在地层网络 |

**身份秘密的核心含义：**
> 林阳不是因为沙暴才觉醒的普通人——他天生就是"接收器"，是父亲遗传给他的体质。他的三阶共振能力是地球这套矿物网络在等待的那个接口。他是"被需要"的，而不是"随机选中"的。这既是他的命运，也是他必须面对的枷锁。

**林阳的内在命题：**
> "我的能力让我更接近沙，还是更接近父亲？"
> "如果成为风暴意识，我还是我吗？"

## 二、女主：沈雨

### 基础信息
- **年龄**：21岁（故事开篇）
- **外形**：长发，清秀，平时穿着普通。末日后剪短头发，眼神从柔软变锐利
- **开篇状态**：大学在读，地质矿物专业，沙暴社区的"矿粒观察者"，用专业知识分析沙粒成分

### 身份弧线
```
大学生（地质专业）
→ 幸存者（父母遇害）
→ 四人团队科学支柱（第30章）
→ 矿物共振频率研究者（第二卷）
→ 寻找林阳矿物化逆转方案的人（第三卷）
```

### 与林阳的感情弧线
```
网络接触（第13章：互动于沙暴社群）
→ 实体相遇（第28章：被救，父母已死）
→ 相依求存（第29-30章：两个失去父母的人）
→ 信任建立（第32-33章：共同调查父亲秘密）
→ 内心靠近（第36章：她发现他能感知次声波）
→ 第一次裂痕（第47章：她一直在悄悄记录他的矿物化数据）
→ 最深的裂缝（第二卷：她知道他的终点，但没有告诉他）
→ 核心对话（终卷前：让他去是为了世界，留住他是为了自己）
```

### 沈雨的核心矛盾
> 她是用科学理解这个世界的人，但林阳是她第一个无法用科学处理的对象。
> 她比任何人都清楚他的矿物化进程意味着什么，却比任何人都更不想接受结论。

## 三、友情线：王磊

### 基础信息
- **年龄**：22岁，男，林阳的老网游好友
- **外形**：偏胖，笑起来爽朗，永远一副没心没肺的样子
- **技能**：会开车，熟悉城市地形，认识各种人，路子广

### 友情弧线
```
游戏里的快乐（第1章）
→ 第一个发消息的人：沙尘暴！！（第2章）
→ 他家被抢，一个背包出现在林阳门口（第14章）——友情线正式开始
→ 父母遇难后陪着，没说一句话（第25章）——最重要的沉默
→ 深夜裂缝：他第一次不装了（第40章）
→ 后期：他的笑越来越少，但每一次都是整个团队最后的温度
```

### 王磊的功能定位
- **情绪晴雨表**：他有多不笑，社会就崩坏了多少
- **对比镜**：他的转变映射出整个世界对普通人的消耗
- **友情线的钩子**：他是第一个真正"坏掉"又"撑起来"的人

## 四、友情线：赵强

### 基础信息
- **年龄**：38岁，男，同栋楼的邻居，前武警，退伍后做保安
- **外形**：高大，短发，脸上一道旧疤，走路带着职业性警觉
- **技能**：实战经验，急救，修车，判断危险的直觉极强

### 友情弧线
```
点头之交（开篇三年）
→ 第一次说话：去备粮，别等人抢（第8章）
→ 带着手摇收音机敲门：一个人没意义（第16章）——正式融入
→ 教林阳用枪（第26章后）："枪是谈判筹码，不是武器。"
→ 成为四人组的安全屏障与决策参谋
→ 赵强的秘密初现（第45章）：他在找妻女，照片被对方摇头
→ 后期：他的道德底线被这个世界一次次逼近
```

### 赵强的隐藏暗线
- 社会崩坏初期，妻子和女儿失散
- 他从不主动提，但每次经过登记点都会去问
- **这是他活着的理由**，也是他绑在旧世界道德准则上的根
- 功能：赵强是林阳的临时父亲，是整个故事里"旧世界的人如何在新世界坚持本心"的代表

---

## 五、暗线人物：沙翁

### 基础信息
- **年龄**：60+岁，男，地表区最老的拾荒者
- **外形**：话少，眼神平静，沙粒扑面也不眨眼，沙好像绕着他走

### 作用
- 认识林阳的父亲，从第一次沙暴就在观察
- 每次出现都是谜语，从不解释，只投放拼图碎片
- 与沈雨的科学证据（遗迹检测）形成两条独立收敛的暗线证据链
- **初次登场**：第38章。直接说："你耳朵里能听见频率，对吗？你父亲也能。"

## 六、第二卷登场人物

### 宋霆（军方派）
- **年龄**：45岁，男，半地下区军事长官
- **目的**：系统性收编所有觉醒者，构建军事控制体系
- **与林阳的关系**：起初林阳只是名单上的一个编号，直到他的三阶能力显现，才成为重点目标
- **核心逻辑**：觉醒者是战略资产，不是人。谁控制觉醒者，谁就控制新世界

### 郑博（科研派）
- **年龄**：55岁，男，核心避城首席科学家，沈雨的导师
- **表面**：反对人体实验的理性主义者
- **真相**：大地呼吸计划的参与者，一直在掩盖真相
- **暗线揭露节点**：第二卷，奚然手握他的把柄，最终自首

### 奚然（资本派）
- **年龄**：38岁，女，核心避城资源委员会代表
- **手牌**：知道林阳父亲的档案，掌握郑博的把柄
- **目的**：实用主义，谁对她有用，她就支持谁
- **揭露节点**：她是告知林阳父亲真相的人

### 沙鸦·向凛（砂行者首领）
- **年龄**：30岁，男，前半地下区工人，右侧脸颊和右臂已完全矿化
- **信仰**：沙暴是进化信号，不是灾难
- **与林阳关系**：亦敌亦友，第一个对林阳说"你是能和沙说话的人"
- **功能**：支线一核心，连接主线、暗线的枢纽人物

### 零七·段岩（实验体）
- **年龄**：28岁，男，皮下矿物纤维可见，力量超强，情绪失控时眼睛发蓝光
- **身份**：人体实验"半成功体"，记忆中保有实验室的关键片段
- **关键信息**：揭露"他们试图制造能进入风暴核心的觉醒者"——林阳是自然成型的接收器，不是人工的
- **悲剧性**：对林阳忠诚，但随时可能情绪失控

## 七、人物关系网

```
林阳
├── 父亲（已失踪/以矿物意识存在）→ 暗线核心，遗传了接收器体质
├── 沈雨 → 感情线，她记录他的矿物化，知道他的终点
├── 王磊 → 友情线，情绪晴雨表，陪他过最难的夜
├── 赵强 → 友情线，临时父亲，教他如何在新世界活着
├── 沙翁 → 暗线桥梁，认识父亲，最早的观察者
├── 沙鸦 → 支线一，第一个告诉他"你和沙在说话"
├── 零七 → 支线二，揭露他是天生接收器的人
├── 宋霆 → 对立，想把他变成军事资产
├── 郑博 → 暗线，造成一切的参与者
└── 奚然 → 告知者，手握父亲档案
```
